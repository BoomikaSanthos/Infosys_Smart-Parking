# Smart Park Backend (Java Spring Boot)

This is the Java Spring Boot implementation of the Smart Park backend, migrated from Node.js/Express.

## Requirements
- Java 17 or higher
- Maven
- MongoDB

## Setup
1. Update `src/main/resources/application.properties` if your MongoDB URI differs.
2. Build the project: `mvn clean install`
3. Run the application: `mvn spring-boot:run`

## API Documentation
The API follows the same contract as the original Node.js backend running on port 5000.
